# Solar Monitor

Дополнение NVDA для радиолюбителей. Обеспечивает быстрый доступ к данным солнечной активности, условиям прохождения на КВ и УКВ через виртуальный клавиатурный интерфейс.

## Возможности

* Солнечные индексы: SFI, число пятен, A-индекс, K-индекс, рентген, солнечный ветер, потоки протонов и электронов, линия гелия, индекс авроры, состояние геомагнитного поля, уровень шума, МПЧ
* Условия прохождения на КВ для диапазонов 80-40М, 30-20М, 17-15М, 12-10М с отдельными значениями для дня и ночи
* Условия прохождения на УКВ: авроральное рассеяние, спорадик для Европы и Северной Америки
* Виртуальный клавиатурный интерфейс без визуальных элементов, разработанный специально для скринридеров
* Контекстные описания параметров по клавише D
* Данные с hamqsl.com, обновляются автоматически с локальным кешем на 15 минут
* Полный интерфейс на русском и английском языках
* Настраиваемый жест через жесты ввода NVDA

## Установка

Скачайте файл `.nvda-addon` со [страницы релизов](https://github.com/R1BQE/solarMonitor/releases) и откройте его. NVDA запросит подтверждение установки и перезапустится.

## Использование

### Назначение жеста

После установки откройте меню NVDA, перейдите в Параметры, затем в Жесты ввода. Найдите категорию "Монитор солнечной активности" и назначьте любой жест на "Открыть или закрыть виртуальный интерфейс монитора солнечной активности".

### Управление клавиатурой

Нажмите назначенный жест для открытия виртуального монитора. NVDA произнесёт "Монитор солнечной активности. Загрузка данных..." и затем озвучит первый параметр после получения данных.

Внутри монитора:

* Стрелка вниз: перейти к следующему параметру
* Стрелка вверх: перейти к предыдущему параметру
* Стрелка вправо: перейти на следующую страницу (Солнечные индексы, Прохождение на КВ, УКВ условия)
* Стрелка влево: перейти на предыдущую страницу
* D: прочитать подробное описание текущего параметра с нормальными значениями и их влиянием на радиосвязь
* Назначенный жест ещё раз: закрыть монитор

### Страницы

**Страница 1: Солнечные индексы**

* SFI: индекс солнечного потока
* Число Вольфа: количество солнечных пятен
* A-индекс: индекс геомагнитной активности за 24 часа
* K-индекс: текущий индекс геомагнитной активности с текстовым статусом
* Рентген: класс рентгеновского излучения Солнца
* Магнитное поле Bz: Z-компонента межпланетного магнитного поля в нанотеслах
* Солнечный ветер: скорость в км/с
* Поток протонов: поток протонов высоких энергий в единицах pfu
* Поток электронов: поток релятивистских электронов в единицах pfu
* Линия гелия: ультрафиолетовое излучение Солнца на длине 304 ангстрема
* Аврора: индекс авроральной активности
* Широта авроры: геомагнитная широта границы авроральной зоны в градусах
* Геомагнитное поле: состояние геомагнитного поля текстом (QUIET, UNSETTLD, ACTIVE, MINOR STORM, MAJOR STORM, SEVERE STORM)
* Уровень шума: уровень шума на КВ по шкале S
* МПЧ: максимально используемая частота

**Страница 2: Прохождение на КВ**

Качество прохождения для каждого диапазона отдельно днём и ночью:

* 80М-40М
* 30М-20М
* 17М-15М
* 12М-10М

Значения: Хорошо, Средне, Плохо.

**Страница 3: УКВ условия**

* Авроральное рассеяние (Северное полушарие)
* Спорадик для Европы, Северной Америки и диапазонов 50/70 МГц

### Описания параметров

Нажмите D на любом параметре для получения подробного описания. Для солнечных индексов: что измеряет параметр, нормальный диапазон, пороговые значения и влияние на радиосвязь. Для КВ и УКВ условий: какие факторы влияют на диапазон и чего ожидать.

## Источник данных

Данные получаются с [hamqsl.com](http://www.hamqsl.com/solarxml.php) - специализированного сервиса данных о Солнце для радиолюбителей. Данные кешируются локально на 15 минут во избежание лишних запросов к серверу.

## Требования

* NVDA 2025.1 или новее
* Windows 10 или новее
* Подключение к интернету для обновления данных
