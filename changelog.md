## 2025.6.25 - Initial release

### Features

* Virtual keyboard interface with no visual elements, designed entirely for screen readers
* Three pages navigated with left/right arrows:
  * **Solar Indices** - SFI, sunspot number, A-index, K-index, X-ray, solar wind, proton and electron flux, helium line, aurora index, geomagnetic field status, noise level, MUF
  * **HF Band Conditions** - propagation quality for 80-40M, 30-20M, 17-15M, 12-10M with separate day and night values
  * **VHF Conditions** - aurora scatter and sporadic-E for Europe (2m, 6m, 4m) and North America
* Up/down arrows to move between parameters on the current page
* D key for context-sensitive parameter descriptions including normal ranges and effect on propagation
* Speech interruption on each keypress - no queue buildup
* Data from hamqsl.com, cached locally for 15 minutes
* English and Russian interface
* Configurable gesture via NVDA Input Gestures (no default assigned)
* Compatible with NVDA 2025.1 and later
